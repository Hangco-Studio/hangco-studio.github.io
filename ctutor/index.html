概要

C语言是一门面向过程的、抽象化的通用程序设计语言，广泛应用于底层开发。C语言能以简易的方式编译、处理低级存储器。C语言是仅产生少量的机器语言以及不需要任何运行环境支持便能运行的高效率程序设计语言。尽管C语言提供了许多低级处理的功能，但仍然保持着跨平台的特性，以一个标准规格写出的C语言程序可在包括类似嵌入式处理器以及超级计算机等作业平台的许多计算机平台上进行编译。正因为C语言的这个特性，它被广泛地应用于各类计算机中。C语言已经是计算机最基础的语言之一了。


第一章 C语言基础

1.1 Hello, world!

请先阅览下面的代码：
#include <stdio.h>
int main() {
  printf("Hello, world!\n");
  return 0;
}
 下面请看对于这段代码的分析：
#include是“导入”，它后面的“<stdio.h>”是一个文件“stdio.h”。这句话的意思就是导入stdio.h的文件内容。这种文件就像个工具箱，里面有各种实用的东西。
int main() {的意思是“‘}’以内的内容是‘主函数’。”主函数是C语言最基本的部分，所有程序都依赖它。
printf("Hello, world!\n");这句话使用了“工具箱”stdio.h中的“printf”函数。这个函数的功能是向屏幕输出内容。里面的"Hello, world!\n"是一个字符串，"\n"是回车。
return 0;的意思是“退出程序”。

这个程序的输出结果：Hello, world!
这就是printf函数输出的内容。
